# 软件介绍

PeachPlayer 是一款采用 Net C#开发的全功能媒体播放器，它基于 `Avalonia ` 强大框架，以清新的薄荷绿为主题，旨在为用户提供流畅的跨平台娱乐体验。

## 🎉 功能亮点

- **广泛兼容**：全面支持 Windows、Mac、Linux 系统（除windows之外系统暂未做适配和测试）。
- **深色模式**：适配夜间使用，保护视力。
- **资源站接入**：轻松管理 json、xml 等格式的 CMS。
- **IPTV 支持**：无缝播放 m3u、genre 格式及集成 EPG。
- **视频平台解析**：解锁主流在线视频内容。
